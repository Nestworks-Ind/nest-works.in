Every step a performance engineer does manually — handled end-to-end in a single run.

01
Dynamic value detection
Compares two recordings to automatically identify session tokens, CSRF tokens, IDs, and timestamps that change between runs.
02
RegexExtractor injection
Generates accurate regex patterns with left-anchor context and injects RegexExtractor elements into the correct sampler — body or header.
03
Response assertions
Adds a ResponseAssertion to every TransactionController — regex-based where extractors exist, body content for the rest.
04
Think time injection
Inserts FlowControlAction pauses between transactions via a single variable — change one value to control all pauses instantly.
05
Header extraction
Detects dynamic values in response headers (Set-Cookie, X-Auth-Token) and sets useHeaders=true automatically.
06
Advanced correlation writing
Implements grouping and match number logic when writing RegexExtractors — handles complex multi-occurrence values accurately.
07
Smart request matching
Matches requests between recordings by URL structure and body similarity — handles JMeter's numeric sampler names correctly.
08
Regex validation
Validates every generated regex against the actual response body before injecting — extractors are guaranteed to work before you open JMeter.
09
Encoded response support
Handles JTL validation when responses are encoded — no correlations missed due to encoding format.
10
Multi-platform app support
Works across Java, Python, and .NET applications — handles response structure differences without any manual configuration.
11
Listener injection
Automatically adds View Results Tree and Aggregate Report listeners at the Test Plan level — ready to monitor execution without any manual setup.

FAQ

Common questions
from engineering teams.

Does NestCraft work with any JMeter script, or only specific frameworks?
NestCraft works with any standard JMeter JMX script — regardless of the application under test, framework, or industry. It has been validated across Java, Python, and .NET applications and works with enterprise platforms in insurance, retirement, and financial services. As long as you have two JMX recordings and their corresponding JTL result files, NestCraft can process them.
Is our test data safe? Do files get uploaded to any server?
Completely safe. NestCraft is a native desktop application — it runs entirely on your machine. Your JMX and JTL files are never uploaded, transmitted, or shared with any external server. Zero bytes of your data leave your environment. This is by design, and it is one of the core reasons the product exists as a desktop application rather than a cloud service.
How long does the onboarding process take?
Most teams are up and running within a single session — typically 30 minutes for installation and configuration, followed by a real use-case validation run against your own JMX files. We walk your team through the process directly. No IT provisioning, no cloud account setup, no lengthy procurement cycles.
What happens if NestCraft misses a correlation or produces incorrect output?
NestCraft validates every generated regex pattern against the actual JTL response body before injecting it — extractors that don't match are not written. This means the output you receive has been verified. In the rare case where a correlation is missed due to an unusual response structure, our support team works directly with you to resolve it — this is part of the engagement model, not a ticketing queue.
We have junior performance engineers — can they use NestCraft effectively?
Yes — this is one of the core value propositions. NestCraft encodes expert-level correlation knowledge into the tool. A junior engineer with basic JMeter familiarity can produce the same output that would previously require a senior engineer with deep scripting experience. The interface is clean and focused, and the guided onboarding ensures your team is confident from day one.
What operating systems does NestCraft support?
NestCraft currently runs on Windows 10 and Windows 11 (64-bit). macOS support is in active development and will be available shortly. If your team requires macOS support urgently, contact us — we can discuss early access options.
