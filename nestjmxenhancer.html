What it does

Everything needed for production‑ready JMeter scripts.

One analysis run. Every correlation, assertion, and timer — automated. No manual steps.

NestJMXEnhancer is a native desktop application — installed on your Windows machine, runs completely offline. No subscription. No cloud infrastructure. No data ever leaves your machine.

Desktop app Zero data stored
01
Dynamic value detection
Compares two recordings to automatically identify session tokens, CSRF tokens, IDs, and timestamps that change between runs.
02
RegexExtractor injection
Generates accurate regex patterns with left-anchor context and injects RegexExtractor elements into the correct sampler — body or header.
03
Response assertions
Adds a ResponseAssertion to every TransactionController — regex-based where extractors exist, body content for the rest.
04
Think time injection
Inserts FlowControlAction pauses between transactions via a single variable — change one value to control all pauses instantly.
05
Header extraction
Detects dynamic values in response headers (Set-Cookie, X-Auth-Token) and sets useHeaders=true automatically.
06
Advanced correlation writing
Implements grouping and match number logic when writing RegexExtractors — handles complex multi-occurrence values accurately.
07
Smart request matching
Matches requests between recordings by URL structure and body similarity — handles JMeter's numeric sampler names correctly.
08
Regex validation
Validates every generated regex against the actual response body before injecting — extractors are guaranteed to work before you open JMeter.
09
Encoded response support
Handles JTL validation when responses are encoded — no correlations missed due to encoding format.
10
Multi-platform app support
Works across Java, Python, and .NET applications — handles response structure differences without any manual configuration.

How it works

From raw recordings to
production-ready JMX in minutes.

Five steps. Real UI. No guessing.

01
Upload Your Recordings

Record 2 scripts with same flow — transaction names and count must match exactly. Drop in two JMX scripts and their JTL result files, one pair per recording. NestJMXEnhancer detects dynamic values across both runs and prepares them for analysis.

2× JMX 2× JTL Drag & drop
02
Configure Your Analysis

Optionally scope which requests to scan, exclude third-party CDN domains, and enable Response Assertions or Think Time pauses — all settings are optional, defaults work out of the box.

Request Filter Domain Exclusions Assertions Think Time
03
Review Correlation Results

The engine surfaces every dynamic value — session tokens, CSRF values, member IDs, claim numbers — with the exact regex, the origin request, and correlation status. No black box. Full visibility.

04
Select & Apply Correlations

Check the correlations you need — or Select All — then click Apply & Export JMX. The tool writes RegexExtractors, Response Assertions, and Think Time pauses directly into your script.

Select all or pick Apply & Export JMX
05
Download & Run in JMeter

Assertions and timers confirmed — click Download JMX. Your production-ready correlated script is ready to open in JMeter and execute. No post-processing. No manual edits needed.
